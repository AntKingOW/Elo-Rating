# OWCS 2024 수집 대상 이벤트 목록

총 35개 A-Tier 이상 이벤트. 시간순으로 스크래핑.
스크래핑 완료 시 [ ] → [x] 체크.

---

## 스크래퍼 사용법

```bash
python liquipedia_scraper.py "<URL>" \
  --region <region> \
  --event-id <event_id> \
  --season 2024 \
  --stage "<stage>" \
  --output <파일명>.csv
```

## 스크래핑 예시 명령어

### Asia Stage 1 Korea (이미 테스트 완료)
```bash
python liquipedia_scraper.py \
  "https://liquipedia.net/overwatch/Overwatch_Champions_Series/2024/Asia/Stage_1/Korea" \
  --region korea --event-id owcs_2024_asia_s1_korea \
  --season 2024 --stage "Stage 1" \
  --output drafts/draft_owcs_2024_asia_s1_korea.csv
```

### FACEIT League S1 EMEA Master
```bash
python liquipedia_scraper.py \
```
